ClassySocial
Socialize with the power of jQuery!

Introduction

ClassySocial is a jQuery plugin written by Marius Stanciu - Sergiu and extended by Gabriel Zanetti. It lets your site visitors easily see what networks you belong to and visit them in a click of a button.

Currently supports Facebook, Google Plus, Twitter, Pinterest, LinkedIn, Dribbble, E-Mail, Socl, Instagram, Vimeo, Youtube, GitHub, Blogger, DeviantArt, Flickr, Skype, Steam, Wordpress and Yahoo out of the box. However, it is possible to define new social networks easily.

Allow your users to see your presence on 18 of the most popular networks instantly with no need of coding knowledge! Have links to your Facebook, Twitter, Vimeo, Dribbble, Instagram, YouTube (and more!) profiles shown in a slick way, sure to impress your users.

Includes functionality to use your current Facebook image as the button that users press to draw out the network buttons, with automatic updating, pulled straight from Facebook! If the social network allows to link the image based on the username then it can also be pulled from any of those networks too.

Setup

First you need to include the jQuery library, since ClassySocial is a plugin. You can download it from the jQuery website or link it directly from the Google CDN.

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>

Secondly, you need to include the jQuery ClassySocial javascript and the CSS file, which you can do it by adding the code below to your page.

<script src="js/jquery.classysocial.min.js"></script>
<link rel="stylesheet" type="text/css" href="css/jquery.classysocial.min.css" />

Next, you implement the structure on which you want to trigger the plugin. Don't to add the classysocial CSS class.

<div class="classysocial"></div>

As the last step, you trigger the plugin on the element you just created. In this case, we trigger it on the element with the class classysocial and set some options. In this case we're displaying 3 buttons (Facebook, Twitter and YouTube)

$('.classysocial').ClassySocial({
    profiles: {
        facebook: {
            handle: 'your-handle'
        },
        twitter: {
            handle: 'your-handle',
            tip: 'Follow me on Twitter!'
        },
        youtube: {
            handle: 'your-handle'
        }
    },
    mainButton: {
        tip: 'Check my profiles out!',
        image: {
            type: 'profile',
            name: 'facebook'
        }
    },
    theme: 'slick',
    orientation: 'arc',
    arcStart: 90,
    arcLength: 180,
    arcRadius: 60
});

In case a new network is needed (suppose Question2Answer), you will need to:

  • Place all the images for the needed themes: images/default/q2a.png, images/slick/q2a.png, images/square/q2a.png
  • Add the needed CSS classes in the css/jquery.classysocial.css file or its minimal version: 
    .classysocial .default .q2a {
    background-image: url(../images/default/q2a.png);
}

.classysocial .slick .q2a {
    background-image: url(../images/slick/q2a.png);
}

.classysocial .square .q2a {
    background-image: url(../images/square/q2a.png);
}
  • Setup the new network in the options of the plugin: 
    $('.classysocial').ClassySocial({
    profiles: {
        facebook: {
            handle: 'your-handle'
        },
        yahoo: {
            handle: 'your-handle'
        },
        q2a: {
            handle: 'your-handle'
            url: 'http://question2answer.org/qa/users/{handle}',
            tip: 'Check out my Q2A profile',
        }
    }
});

Options

  • Parameters

  • theme - the current theme, can be default, square or slick. Default value is default
  • orientation - orientation, can be arc, line or linedown, default is arc. Controls whether the social network images are displayed as an arc around the main button, in a horizontal line or in a vertical line
  • arcStart - start of the arc, from 1 to 360, default is 0
  • arcLength - length of the arc, from 1 to 360, default is 360
  • arcRadius - the radius of the arc, default is 80
  • gap - the gap (in pixels) between the profile network images, default is 80 pixels
  • profiles - it's an object that can contain any of these values: facebook, google, twitter, pinterest, linkedin, dribbble, email, socl, instagram, vimeo, youtube, github, blogger, deviantart, flickr, skype, steam, wordpress and yahoo. Each of them are themselves objects that are described below
  • profiles.socialNetwork - object that contains the following data for the social network: handle, url, tip, imageUrl. Note the handle is the only mandatory field needed to display the social network image. These fields are described below
  • profiles.socialNetwork.handle - handle/username for the given social network
  • profiles.socialNetwork.url - the URL for the social network that is constructed based on the handle. For example, in the case of Facebook, this is the default URL: http://www.facebook.com/{handle} where {handle} is replaced by the option with the same name
  • profiles.socialNetwork.tip - tooltip displayed when hovering the social network image. For example, in the case of Facebook, this is the default value: View my Facebook page
  • profiles.socialNetwork.imageUrl - the URL from which the main button image can be replaced. For example, in the case of Facebook, this is the default URL: https://graph.facebook.com/{handle}/picture?type=large. This allows to fetch the profile image of the user instead of using the main button default image. Currently, the only social network that supports this feature is Facebook but it is possible to support this feature by just adding this option to any profile object. Note this works together with main button image options explained below
  • mainButton - it's an object that can contain any of these values: tip and image
  • mainButton.tip - tooltip displayed when hovering the main button. The default value: Find me here!
  • mainButton.image - it's an object that can contain any of these values: type and name
  • mainButton.image.type - it's a string that can be profile or default. The first one allows the to fetch the main button image from a network usingt the image URL set for that network. The last one forces the main button image to be the default image set in the CSS file
  • mainButton.image.name - it's a string that can be any of the social networks passed in the options. It determines which of the social networks will be used to replace the main button image. If the image URL of the given social network is not set in the profile then the image displayed will be the default one
